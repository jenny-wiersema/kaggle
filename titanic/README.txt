This project is broken down into three python files:

1) titanic_feature_investigation.py: code to run investigate possible features for use in the logistic regression.
2) titanic_logistic.py: code to run a logistic regression, using the features investigated in titanic_feature_investigation.py
3) titanic_utils.py: all classes and functions needed for titanic_feature_investigation.py and titanic_logistic.py
